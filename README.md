## Installation

1. Clone the repository:

```
git clone https://github.com/Makksaw/novapost-web.git
```

2. Navigate to the project directory:

```
cd novapost-web
```

3. Install the dependencies:

```
npm install
```

## Run the project

```
npm start
```

[localhost:3000](http://localhost:3000)
